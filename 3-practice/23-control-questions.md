## Глава 23. Метод контрольных вопросов для генерации гипотез

Для поиска нестандартных гипотез используйте контрольные вопросы. Ищите, как можно:
- **Адаптировать**
- **Исключить**
- **Изменить**
- **Сделать наоборот**
- **Скомбинировать**
- **Заменить**
- **Изменить порядок**

*Пример ограничения:* «Мы недостаточно делаем звонков клиентам»

- **Сделать наоборот:** «Поменять подход с активного звонка на пассивный» — пригласить клиентов звонить нам
- **Изменить порядок:** «Вместо звонка после первого контакта сначала отправлять полезные материалы, а затем звонить»

В результате появляется несколько десятков гипотез, из которых выбираем 2–3 простым голосованием.
