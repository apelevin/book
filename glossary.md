# ПРИЛОЖЕНИЕ. Глоссарий Правократии

| Термин | Определение |
|--------|-------------|
| **Круг** | Команда до 10 человек, организованная вокруг общей цели. Заменяет отдел/департамент |
| **Роль** | Динамическая функция с определёнными действиями и ожидаемым результатом. Заменяет должность |
| **Лидер** | Фасилитатор, создающий условия для автономной работы команды. Не руководитель |
| **Ограничение** | Барьер, мешающий достижению цели. Не упрёк, а точка роста |
| **Гипотеза** | Смелое предположение по устранению ограничения, которое можно протестировать |
| **OKR** | Objectives and Key Results — инструмент фокусировки на изменениях |
| **HADI** | Hypothesis — Action — Data — Insight — фреймворк для работы с гипотезами |
| **Консент** | Принятие решения при отсутствии обоснованных возражений. Не консенсус |
| **Peer Review** | Обратная связь от коллег по запросу |
| **Паспорт ограничения** | Документ с описанием ограничения, фактами и данными |
| **Тактический круг** | Временная команда для решения конкретной задачи |
| **FTE** | Full-Time Equivalent — единица измерения загрузки роли |
| **ExtraTime** | Северная звезда компании — сколько времени мы высвобождаем людям |
| **Beyond Budgeting** | Управленческий фреймворк адаптивного финансового управления |
| **Относительная оценка** | Подход к вознаграждению, основанный на сравнении с коллегами и собственной динамикой, а не на выполнении фиксированного плана |
| **Gaming** | Деструктивное поведение, при котором команды оптимизируют собственный бонус вместо бизнес-результата — через занижение целей, сокрытие сделок и выбор лёгких метрик |
| **Экономический бенчмарк** | Ориентир экономической привлекательности направления. Не порог для бонуса, а контекст для понимания позиции команды |
| **«Муха на стене»** | Метод наблюдения за командой без вмешательства |
| **Теневое наблюдение** | Сопровождение коллеги в течение дня для понимания реальной работы |
| **Карта социальных связей** | Визуализация реальных связей между людьми в кругах |
| **Когнитивная карта** | Рисунок, показывающий, как человек видит структуру компании |
| **«Нарисуй опыт»** | Визуализация ощущений через рисование для обхода словесных фильтров |
| **Словесные ассоциации** | Диагностика отношения к ключевым понятиям культуры |
| **Анализ потоков** | Визуализация движения информации и решений между кругами |
| **Сценарий** | Конкретная история, проверяющая гипотезу на реалистичность |
| **«Заголовки будущего года»** | Упражнение визионирования: какие заголовки мы хотим увидеть через год |
| **Быстрый прототип процесса** | Минимальная версия решения для быстрой проверки |
| **Культурный дневник** | Набор для самонаблюдения сотрудников на 1-2 недели |
